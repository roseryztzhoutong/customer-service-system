# 交付部知识库管理系统

这是一个轻量级的 AI 客服知识库管理系统，用于教育培训公司交付部管理学员常见问题和智能回答。


## 功能

- **📝 问题提交** - 老师和客服快速记录学员常见问题
- **💬 AI 客服助手** - 基于知识库智能匹配回答
- **📚 知识库管理** - 查看、搜索、筛选、导出问答
- **📊 数据统计** - 知识库总量、分类分布、团队贡献
